# DracoBits

Demo final desenvolvida para a disciplina de Design e Desenvolvimento de Jogos.

## Como executar

Abra `index.html` em um navegador moderno ou publique a raiz do projeto no GitHub Pages.

## Controles

- Mouse: interagir com botões, fragmentos e cristais.
- Tecla `M`: ligar ou desligar a música e os efeitos sonoros.

## Objetivo

Mantenha Draco saudável e restaure o Núcleo Arcadia até 100% de estabilidade.
