# DracoBits — Presentation Edition

Demo final do jogo DracoBits, desenvolvida com HTML5 Canvas, CSS, JavaScript modular e Web Audio API.

## Destaques

- Quatro fases com dificuldade crescente
- Necessidades de Draco: fome, higiene, felicidade e energia
- Fragmentos corrompidos, cristais e combos
- Música ambiente adaptativa
- Condições de vitória e derrota
- Tutorial, classificação indicativa, créditos e telas finais

Abra `index.html` em um navegador moderno ou publique o conteúdo no GitHub Pages.
